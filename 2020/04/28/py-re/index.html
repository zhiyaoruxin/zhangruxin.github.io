关于python中正则表达式的使用整理一波

import re

常用

  1. 正则表达式 - findall方法
    re.findall(匹配规则，原始文本) 得到的是一个列表
     \d 表示匹配一个数字  \d\d\d表示匹配三个数字
  2. 正则表达式 - 非贪婪匹配（.*?）
正则表达式 描述
\d 匹配一个数字字符
\w 匹配一个字母、数字及下划线字符
\s 匹配一个空白字符
\S 匹配一个非空白字符
\n 匹配一个换行符
\t 匹配一个制表符
. 匹配一个任意的字符，换行符除外
* 匹配前面的字符无限次直到停止
? 常和.与*配合使用，组成非贪婪匹配
() 匹配括号内的表达式，也表示一个组
^ 匹配行的开头
\A 匹配字符串开头

  1. 非贪婪匹配.*?

         res = '<h3 class="c-title"><a href="http://news.hexun.com/2018-12-12/195521925.html" data-click="{一堆英文}"><em>阿里巴巴</em>电商脱贫成“教材” 累计培训逾万名县域干部</a>'
     p_title = '<h3 class="c-title">.*?>(.*?)</a>'
     result3 = re.findall(p_title,res)
     print(result3)

  2. 正则表达式 - 自动考虑换行
    re.findall(匹配规则，原始文本，re.S)

  3. 正则表达式 - 小知识点
    sub()方法：
     re.sub(需要替换的内容，替换值，源字符串)
    []中括号的用法
     1. 在中括号里的内容不再有特殊含义
     company = '*华能信托'
     company = re.sub('[*]',company)
     print(company)
 2. 表示某几个的范围，比如把日期中的年月日换成-号
     date = '2018年12月12日'
     date = re.sub('[年月日]'，'-',date)
     date = date[:-1]
     print(date)
     date = date[:-1]其实是一个字符串切片，这个的含义就是取该字符串除了最后一个元素的其他元素
    数据清洗并打印输出
     清理新闻标题
     字符串.strip()函数把不需要的空格和换行符去掉

re模块常量

  • re模块中有9个常量，常量的值都是int类型
常量 简写 描述
re.IGNORECASE re.l 进行忽略大小写匹配
re.ASCII re.A 让\w,\W,\b,\B,\d,\D,\s,\S只匹配ASCII，而不是Unicode
re.DOTALL re.S DOT表示.,ALL表示所有，连起来就是.匹配所有，包含换行符\n,默认模式下，是不能匹配换行符\n的
re.MULTILINE re.M 多行模式，当某个字符串中有换行符\n，默认模式下是不支持换行符特性的，比如行开头和行结尾，而多行模式下是支持匹配行开头的
re.VERBOSE re.X 详细模式，可以在正则表达式中加注解
re.LOCALE re.L 由当前语言区域决定\w,\b和大小写敏感匹配，这个标识只能对byte样式有效
re.UNICODE re.U 与ASCII模式类似，匹配Unicode编码支持的字符
re.DEBUG 显示编译时的debug信息
re.TEMPLATE re.T 禁用回溯
  • 常量总结：
    1. 9个常量中，前5个（IGNORECASE,ASCII,DOTALL,MULTILINE,VERBOSE）有用处，有2个（LOCALE,UNICODE）官方不建议使用，2个（TEMPLATE,DEBUG）实验性功能，不能依赖
    2. 常量在re常用函数中都可以使用。
    3. 常量可叠加使用，因为常量值都是2的幂次方，叠加时使用|符号，不能使用+符号 如：re.findall(pattern,text,re.IGNORECASE|re.MULTILINE)

re模块函数

re模块中有12个函数

  1. 查找一个匹配项
    1. search： 查找任意位置的匹配项
    2. match： 必须从字符串开头匹配
    3. fullmatch： 整个字符串与正则完全匹配
      案例：
       import re
 text = 'a猪哥大帅b，猪哥大帅b'
 pattern = r'猪哥大帅b'
 print(re.search(pattern,text).group())  # 猪哥大帅b
 print(re.match(pattern,text))  # None
 print(re.fullmatch(pattern,text)) # None
      注： 查找一个匹配项，返回的都是一个匹配对象（Match）
  2. 查找多个匹配项
    1. findall： 从字符串任意位置查找，返回一个列表
    2. finditer： 从字符串任意位置查找，返回一个迭代器
      案例：
       import re
 text = 'a猪哥大帅b，猪哥大帅b'
 pattern = r'猪哥大帅b'
 print(re.findall(pattern,text)) # ['猪哥大帅b', '猪哥大帅b']
 print(list(re.finditer(pattern,text))) # [<re.Match object; span=(1, 6), match='猪哥大帅b'>, <re.Match object; span=(7, 12), match='猪哥大帅b'>]
      注： 如果可能存在大量的匹配项的haul，建议使用finditer函数，一般情况使用findall函数
  3. 分割
    re.split(pattern,string,maxsplit=0,flags=0)函数：
     pattern分开string
 maxsplit表示最多进行分割次数
 flags表示模式
    案例：
         import re
     text = 'a猪哥大帅b，猪哥大帅b'
     pattern = r'，'
     print(re.split(pattern,text,maxsplit=1,flags=re.IGNORECASE))  # ['a猪哥大帅b', '猪哥大帅b']
    注： 在不需要正则支持且数据量和数次不多的情况下使用str.split函数更合适，反之使用re.split函数
  4. 替换
    1. re.sub(pattern,repl,string,count=0,flags=0)
      repl替换掉string中被pattern匹配的字符
      count表示最大替换次数
      flags表示正则表达式的常量
    2. re.subn(pattern,repl,string,count=0,flags=0) 返回一个元组（字符串，替换次数）
  5. 编译正则对象
    compile函数和template函数 将正则表达式的样式编译成一个正则表达式对象
    案例：
         import re
     text = 'a猪哥大帅b，猪哥大帅b'
     pattern = r'猪哥大帅b'
     pattern_obj = re.compile(pattern)
     print(pattern_obj.search(text).group())
  6. 其他
    re.escape(pattern) 可以转义正则表达式中具有特殊含义的字符，如.或*
    案例：
         import re
     text = 'a猪哥.大帅b，猪哥*大帅b'
     pattern = re.escape('猪哥.大帅')
     print("转义后的pattern："+pattern ) # 转义后的pattern：猪哥\.大帅
     print(re.search(pattern,text).group())  # 猪哥.大帅
    re.purge()函数只用就是清除正则表达式缓存
